go-http-routers-testing
======

Go 路由的性能测试工具

https://caixw.github.io/go-http-routers-testing/



### 安装

```shell
go get github.com/caixw/go-http-routers-testing
```

##### 运行测试

可以使用 test.benchtime 指定每个测试的运行时间。
```shell
go run main.go -test.benchtime=5s
```
